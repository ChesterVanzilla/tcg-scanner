# CardScan CM – Update auf Version 4

## Zu ersetzende Dateien

Lade diese vier Dateien in das Hauptverzeichnis deines bestehenden GitHub-Repositorys und ersetze die vorhandenen Versionen:

- `index.html`
- `styles.css`
- `app.js`
- `service-worker.js`

Die übrigen Dateien können unverändert bleiben.

## GitHub-Schritte

1. Repository öffnen.
2. `Add file → Upload files` auswählen.
3. Die vier Dateien aus dem Update-Ordner hineinziehen.
4. Unten `Commit changes` auswählen.
5. Etwa zwei Minuten auf die neue GitHub-Pages-Bereitstellung warten.

## Update auf dem iPhone erzwingen

Öffne die Website zuerst direkt in Safari und ergänze `?v=4` am Ende, zum Beispiel:

```text
https://DEIN-NAME.github.io/pokemon-card-scanner/?v=4
```

Danach:

1. Safari-Seite einmal neu laden.
2. Die installierte Home-Bildschirm-App vollständig schließen.
3. Die App erneut öffnen.
4. Eine Karte scannen.
5. `Erkennungsdetails anzeigen` öffnen.

Ganz oben muss stehen:

```text
CardScan CM v4.0
```
